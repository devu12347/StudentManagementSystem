
# Student Management System

A simple Student Management System developed using Python and SQLite.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* Store data using SQLite Database

## Technologies Used

* Python
* SQLite3

## Project Structure

StudentManagementSystem/

├── main.py

├── student.db

└── README.md

## How to Run

1. Install Python.
2. Download or clone the repository.
3. Open the project folder in Terminal or Command Prompt.
4. Run the following command:

python main.py

4. Update Student
